世界各国の人口ピラミッド

ウガンダから日本まで、平均年齢の順に各国の人口ピラミッドを3Dに並べた

From Uganda to Japan

Population pyramids in each country

This visualisation is 3D data visualisation containing 2D population pyramid graphs of 201 countries (regions) on the planet. Pyramids are ordered by average age, from Uganda as the youngest to Japan as the oldest. You can rotate view and zoom in / out by dragging and using the mouse scroll. When hovering on each pyramid, you can see the country's 2D pyramid on the bottom­ left side.

This work was submitted as a class assignment.

Data source: UN World Population Prospects 2017

Languages and libraries: JavaScript (Three.js, jQuery), SQL for data acquisition and process
