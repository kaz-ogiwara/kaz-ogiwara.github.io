140 years temperature in Tokyo

Daily average temperature in Tokyo over the past 140 years

This visualisation is a heat map which describes how daily average temperature in Tokyo has changed over the past 140 years, from 1886 to 2015. Each dot, which can be divided into nine colours from blue to red, describes an average temperature on each day. By hovering each dot, you can check the correct temperature on that day. The story of this visualisation is that Tokyo is gradually becoming hotter: We can see the summer is getting hotter and longer, and, in contrast, winter is getting warmer and shorter.

This work was submitted as a class assignment.

Data source: Japan Meteoroligical Agency

Languages and libraries: JavaScript (jQuery), SQL (MySQL) and PHP for retrieving and shaping data
